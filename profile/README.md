## 프로젝트 개요

> 사용자의 이력 정보를 기반으로 이력서 작성 · 포트폴리오 탐색 · 디지털 명함 공유까지
> 하나의 흐름으로 제공하는 인터랙티브 웹 애플리케이션입니다.

HTML · CSS · JavaScript만으로 구현했으며, **로컬스토리지 데이터 저장**,
**PDF 다운로드**, **동적 UI 생성** 등의 기능을 제공합니다.

---

## 주요 기능

| 페이지 | 설명 |
|:--|:--|
| **메인** | 전체 흐름 안내 및 진입점 |
| **이력서** | 입력 폼 기반 자동 정리 · PDF 다운로드 |
| **포트폴리오** | 프로젝트를 카드로 탐색 |
| **디지털 명함** | 명함 제작 및 공유 |
